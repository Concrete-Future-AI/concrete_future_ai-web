# 外贸电商AI转型落地页 - 项目说明

## 📋 项目概览

本项目为"炬象未来"官网的外贸电商AI数智化转型服务落地页，采用现代简约的Anthropic风格设计，提供完整的AI转型咨询和培训服务介绍。

---

## 🎨 设计特点

### 1. 设计风格
- **参考**：Anthropic简约现代风格
- **配色**：#F5F5F0 (浅米白) + #F97316 (行动橙) + #1A1A1A (深黑)
- **字体体系**：
  - Display: Space Grotesk (几何、现代)
  - Body: IBM Plex Sans (专业、易读)
  - Mono: JetBrains Mono (数据、代码)
  - 中文: Noto Sans SC (思源黑体)

### 2. 页面结构（8个屏幕）
```
1. 英雄区 (Hero Section)          ✅ 已完成
   - 震撼视觉 + 核心价值主张

2. 痛点共鸣区 (Pain Points)       ✅ 已完成
   - 传统 vs AI化对比表格

3. 解决方案架构 (Solutions)       ✅ 代码已提供
   - 深度咨询 + 培训体系

4. 课程模块详解 (Courses)        📝 设计文档已有
   - 手风琴式10大模块展开

5. 客户案例与成果 (Case Studies)  📝 设计文档已有
   - 数据化呈现转型效果

6. 服务流程 (Service Flow)       📝 设计文档已有
   - 从咨询到落地的清晰路径

7. 价格价值锚定 (Pricing)        📝 设计文档已有
   - 投资回报率计算

8. 行动召唤 (CTA)                📝 设计文档已有
   - 预约免费诊断
```
