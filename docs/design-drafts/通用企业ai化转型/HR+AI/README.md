
  # 人才引擎 TalentHub - HR+AI 智能人力资源平台

  **客户案例**: 某大型科技企业（5000+员工）AI人力资源管理实践
  
  ## 📊 项目概述
  
  人才引擎是一套以人为本的AI驱动HR管理平台，通过智能招聘、绩效评估、员工成长和组织健康分析，打造温暖而高效的人才管理体系。
  
  ### 核心价值
  
  - ✅ **招聘效率提升65%** - AI智能简历筛选和候选人匹配
  - ✅ **员工留存率提升40%** - 个性化成长路径和发展建议
  - ✅ **绩效评估公平性提升85%** - 数据驱动的客观评估体系
  - ✅ **入职体验满意度95%** - 智能化新员工引导和培训
  
  ## 🚀 技术栈
  
  - **前端框架**: React 18 + TypeScript + Vite
  - **UI组件**: Tailwind CSS + shadcn/ui
  - **动画**: Framer Motion（流畅的成长动画）
  - **状态管理**: React Hooks
  - **字体**: Poppins（友好圆润）+ IBM Plex Mono（数据展示）
  
  ## 💡 核心功能
  
  1. **组织健康看板** - 实时员工状态和组织诊断
  2. **AI智能招聘台** - 智能简历解析和候选人匹配
  3. **绩效与晋升** - 公正的绩效评估和晋升建议
  4. **智能入职系统** - 温暖的新员工体验管理
  5. **员工成长罗盘** - 个性化职业发展路径规划
  6. **AI洞察中心** - 人才趋势和组织优化建议
  
  ## 🎨 设计特色
  
  - **温暖琥珀+翡翠绿配色** - 体现人性化和成长理念
  - **柔和圆角设计** - 营造友好温馨的氛围
  - **生长动画** - 象征员工成长和发展
  - **Poppins字体** - 现代友好的视觉体验
  
  ## 📦 安装和运行
  
  ```bash
  # 安装依赖
  npm install
  
  # 启动开发服务器
  npm run dev
  
  # 构建生产版本
  npm run build
  ```

  - 员工满意度: 从76%提升至92%
