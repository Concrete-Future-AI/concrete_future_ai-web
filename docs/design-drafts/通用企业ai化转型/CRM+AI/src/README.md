# 销速引擎 (Velocity AI)

> 现代化AI销售协作中心 - 让销售更智能，让团队更高效

## 📖 项目简介

**销速引擎 (Velocity AI)** 是一个B2B企业级AI销售赋能与CRM平台，采用现代协作中心的设计理念，类似Monday.com和Notion的设计风格。系统通过AI驱动的Velocity Score评分系统，帮助销售团队识别高价值线索，提升转化率，优化销售流程。

### 🎯 核心价值

- **AI智能评分** - 基于多维度数据的Velocity Score，实时评估线索质量
- **协作优先** - 内置评论、@提及、活动流，团队信息透明共享
- **灵活视图** - 支持列表、看板、日历、表格四种视图，适应不同工作习惯
- **可解释AI** - 清晰展示AI决策依据，建立信任
- **实时洞察** - 持续监控客户行为，提供及时建议
- **分级权限** - VP/Manager/Rep三级角色权限管理
- **AI助手** - 24/7智能对话助手，随时提供帮助

## 🎯 使用场景

### 场景1: 线索优先级排序
客户经理打开智能线索工作台，系统根据Velocity Score自动将最有价值的线索排在前面，并提供AI建议的下一步行动。

### 场景2: 团队协作跟进
销售经理在某个重要交易下@提及团队成员，大家在评论区讨论策略，所有人实时看到更新。

### 场景3: 业绩预测分析
VP在销售作战室查看AI预测，系统显示本季度有望超额完成目标8%，并建议提前规划下季度资源。

### 场景4: 风险客户识别
系统自动识别出3天未互动的高价值客户，发送预警通知，并建议立即电话跟进。
