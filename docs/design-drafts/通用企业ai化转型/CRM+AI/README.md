
  # 销速引擎 SalesTurbo - CRM+AI 智能协作平台

  **客户案例**: 某世界500强制造企业AI化转型实践
  
  ## 📊 项目概述
  
  销速引擎是一套基于AI的现代化CRM协作平台，通过智能化的销售流程管理、实时数据洞察和AI辅助决策，帮助销售团队提升78%的工作效率。
  
  ### 核心价值
  
  - ✅ **销售效率提升78%** - AI自动化任务分配和跟进提醒
  - ✅ **成交率提升35%** - 智能商机评分和预测分析
  - ✅ **客户响应速度提升3倍** - 实时通知和协作机制
  - ✅ **数据准确性提升90%** - 自动数据采集和更新
  
  ## 🚀 技术栈
  
  - **前端框架**: React 18 + TypeScript + Vite
  - **UI组件**: Tailwind CSS + shadcn/ui
  - **动画**: Framer Motion
  - **状态管理**: React Hooks
  - **字体**: Outfit + JetBrains Mono（专业数据展示）
  
  ## 💡 核心功能
  
  1. **销售作战室** - 实时销售漏斗和团队协作
  2. **智能线索工作台** - AI驱动的线索评分和分配
  3. **交易洞察室** - 深度商机分析和成交预测
  4. **预测与分析中心** - 销售预测和数据看板
  5. **AI功能中心** - 邮件生成、会议准备、剧本推荐等
  6. **权限管理** - 细粒度的角色权限控制
  
  ## 🎨 设计特色
  
  - **深蓝+电光青配色** - 传达专业性和速度感
  - **数据流动动画** - 强化实时性和智能化
  - **专业字体组合** - Outfit现代感 + JetBrains Mono数据展示
  - **不对称卡片布局** - 突出重点信息和数据
  
  ## 📦 安装和运行
  
  ```bash
  # 安装依赖
  npm install
  
  # 启动开发服务器
  npm run dev
  
  # 构建生产版本
  npm run build
  ```
  
  ## 👥 角色演示
  
  系统支持三种角色视角切换：
  - **VP张（销售副总裁）** - 战略层面的数据分析和团队管理
  - **李经理（销售经理）** - 团队业绩和商机管道管理
  - **小王（客户经理）** - 日常客户跟进和任务执行
