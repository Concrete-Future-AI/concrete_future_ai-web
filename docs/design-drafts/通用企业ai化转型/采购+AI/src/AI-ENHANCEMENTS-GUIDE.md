## 🎨 Material Design主题系统

### 设计原则

遵循Google Material Design 3.0，并进行深度定制：

1. **层次感** - Elevation阴影系统
2. **流动性** - Material Motion动画
3. **响应式** - 触觉反馈和状态变化
4. **可访问性** - 高对比度和清晰层级

### 配色系统

```css
/* 主色调 - 深蓝主题 */
--md-primary: #1565C0
--md-primary-variant: #0D47A1
--md-secondary: #00897B

/* AI主题色 - 紫色渐变 */
--ai-primary: #6366F1
--ai-secondary: #8B5CF6
--ai-gradient: linear-gradient(135deg, #6366F1, #8B5CF6)

/* 语义化颜色 */
--md-success: #00897B
--md-warning: #F57C00
--md-error: #C62828
```

### Elevation系统

**5级阴影层次**，创造深度感：

```css
elevation-1  /* 轻微悬浮 - 卡片 */
elevation-2  /* 中等悬浮 - 重要卡片 */
elevation-3  /* 明显悬浮 - 顶部导航 */
elevation-4  /* 强烈悬浮 - 对话框 */
elevation-5  /* 最高层级 - 模态层 */
```

### 交互动画

**Material Motion标准**：

```css
/* 过渡时长 */
--motion-duration-short: 150ms   /* 快速反馈 */
--motion-duration-medium: 250ms  /* 标准过渡 */
--motion-duration-long: 350ms    /* 复杂动画 */

/* 缓动函数 */
--motion-easing-standard: cubic-bezier(0.4, 0.0, 0.2, 1)
--motion-easing-decelerate: cubic-bezier(0.0, 0.0, 0.2, 1)
--motion-easing-accelerate: cubic-bezier(0.4, 0.0, 1, 1)
```

### 特殊效果

#### 1. AI光晕效果
```css
.ai-glow        /* 柔和光晕 */
.ai-glow-strong /* 强烈光晕 */
```

#### 2. 玻璃态效果
```css
.glass-effect      /* 毛玻璃背景 */
.glass-effect-dark /* 深色毛玻璃 */
```

#### 3. 悬浮卡片
```css
.hover-lift /* 鼠标悬停上浮 */
```

#### 4. 数据流动画
```css
.data-flow /* AI数据流动效果 */
```
