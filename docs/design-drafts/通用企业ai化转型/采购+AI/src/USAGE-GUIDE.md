# 智采云使用指南

## 🚀 快速开始

### 1. 切换角色
在顶部导航栏选择不同的用户角色：
- **采购总监 (王总)** - 战略决策视图
- **采购专员 (李明)** - 日常工作视图  
- **业务申请人 (张悦)** - 购物式申请视图

### 2. 使用AI助手
点击右下角的 **AI助手浮动按钮** (紫色渐变圆形按钮)：
- 输入问题或需求
- AI会根据您的角色提供针对性建议
- 点击建议按钮快速执行操作

### 3. 探索AI功能

#### 📊 采购总监体验
**AI智能洞察**
- 查看集中采购机会
- 获取供应商整合建议
- 了解季度预算趋势

**AI预测分析**
- Q3采购支出预测（置信度87%）
- IT设备类预测（置信度92%）
- 市场服务类预测（置信度84%）

**AI风险监控**
- 实时查看供应商中断风险
- ESG合规预警
- 价格波动提醒
- 综合风险评分：72/100

**操作建议**：
1. 点击"导出报告"生成AI分析报告
2. 点击洞察卡片的"查看详情"深入了解
3. 监控高风险供应商（红色标识）

#### 💼 采购专员体验
**AI智能提醒**
- 注意顶部的AI提醒横幅
- 显示价格低点采购机会
- 点击"采纳建议"一键创建采购单

**AI工作建议**
- 价格低点机会（高影响）
- 合同到期提醒（中影响）
- 自动化审批建议
- 供应商表现预警

**供应商健康度监控**
- 实时查看供应商健康分
- 绿色(85+): 健康
- 橙色(70-84): 良好
- 红色(<70): 需关注

**任务管理**
- AI智能优先级排序
- 红色Badge = 高优先级
- 点击"完成"标记任务完成

**操作建议**：
1. 优先处理"高优先级"和"今天到期"任务
2. 关注健康分<70的供应商
3. 使用AI建议优化工作流程

#### 🛒 业务申请人体验
**AI智能搜索**
- 在顶部搜索框输入需求
- AI会理解自然语言
- 例如："我需要一台笔记本电脑用于设计工作"

**AI个性化推荐**
- 查看"AI为您推荐"卡片
- 基于您的工作习惯
- 90%同事推荐的商品
- 限时优惠提醒

**快速申请**
- 点击四个快速模板之一
- AI自动填充常用信息
- 减少40%填写时间

**订单追踪**
- 查看申请进度条
- 绿色勾选 = 已完成
- 蓝色时钟 = 进行中
- 灰色圆点 = 未开始

**AI加速标识**
- 绿色Badge显示AI节省的时间
- 例如："AI加速+2天"

**一键复购**
- 在"常购商品"区域
- 点击"快速复购"
- AI自动创建采购申请

**操作建议**：
1. 使用快速模板减少填写时间
2. 查看AI推荐获取优惠信息
3. 一键复购常用商品
4. 实时追踪订单状态

## 🎨 界面元素说明

### 颜色语义
- **深蓝 (#1976D2)**: 主要功能按钮
- **紫蓝渐变 (#6366F1 → #8B5CF6)**: AI相关功能
- **绿色**: 积极/成功/健康
- **橙色**: 警告/中等风险
- **红色**: 危险/高风险/紧急

### 徽章(Badge)说明
- 🟢 **绿色**: 已通过/低风险/健康
- 🟠 **橙色**: 处理中/中风险/注意
- 🔴 **红色**: 审批中/高风险/警告
- 🔵 **蓝色**: 一般信息
- 🟣 **紫色渐变**: AI驱动功能

### 图标含义
- ✨ **Sparkles**: AI功能
- 🧠 **Brain**: AI分析
- 🛡️ **Shield**: 安全/风险监控
- ⚡ **Zap**: 快速/加速
- 🤖 **Bot**: AI助手
- 💡 **Lightbulb**: 建议
- 📈 **TrendingUp**: 上升趋势
- ⚠️ **AlertTriangle**: 警告

## 💡 使用技巧

### 提高效率
1. **善用AI助手**: 遇到不确定的操作，直接问AI
2. **关注AI提醒**: 顶部横幅的AI提醒通常很重要
3. **一键操作**: 使用"快速复购"等一键功能
4. **批量处理**: 在专员视图按优先级批量处理任务

### AI功能最佳实践
1. **采购总监**: 每周查看AI风险监控和预测分析
2. **采购专员**: 每天查看AI工作建议，优先处理高影响项
3. **申请人**: 使用AI推荐获取最佳采购时机和价格

### 数据解读
**风险评分**:
- 80-100: 健康（绿色）
- 60-79: 良好（黄色）
- 40-59: 注意（橙色）
- 0-39: 警告（红色）

**AI置信度**:
- 90%+: 高度可信
- 80-89%: 可信
- 70-79%: 参考
- <70%: 仅供参考

## 🎯 常见场景

### 场景1: 紧急采购
1. 切换到"申请人"角色
2. 使用AI搜索找到商品
3. 点击快速模板创建申请
4. AI会自动加速审批流程
5. 实时追踪订单状态

### 场景2: 供应商风险管理
1. 切换到"总监"角色
2. 查看AI风险监控卡片
3. 关注红色"严重"和橙色"高风险"项
4. 点击"查看详情"了解建议
5. 点击"监控"启动持续跟踪

### 场景3: 日常任务管理
1. 切换到"专员"角色
2. 查看AI排序后的任务列表
3. 优先处理红色高优先级任务
4. 点击"完成"标记已处理
5. 查看AI工作建议优化流程

### 场景4: 战略决策
1. 切换到"总监"角色
2. 查看AI智能洞察卡片
3. 分析AI预测的Q3支出趋势
4. 导出AI分析报告
5. 基于AI建议调整采购策略
