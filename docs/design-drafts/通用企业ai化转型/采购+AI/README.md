# 采购智脑 ProcureMind - 采购+AI 智能采购管理平台

**客户案例**: 某大型制造企业（年采购额50亿+）AI采购管理实践

## 📊 项目概述

采购智脑是一套以智能决策为核心的AI采购管理平台，通过供应商智能评估、需求预测、价格优化和风险管控，实现采购的降本增效。

### 核心价值

- ✅ **采购成本降低18%** - AI驱动的议价和批量优化
- ✅ **采购周期缩短40%** - 智能需求预测和自动化流程
- ✅ **供应商质量提升30%** - AI多维度评估和动态管理
- ✅ **库存成本降低25%** - 精准需求预测和JIT采购

## 🚀 技术栈

- **前端框架**: React 18 + TypeScript + Vite
- **UI组件**: Tailwind CSS + shadcn/ui
- **数据可视化**: D3.js + Recharts
- **状态管理**: React Hooks + Context
- **字体**: Work Sans（现代简洁）+ IBM Plex Mono（数据）

## 💡 核心功能

1. **智能看板** - 采购全景视图和关键指标
2. **供应商管理** - AI评估和动态评分系统
3. **需求预测** - 基于历史和市场的智能预测
4. **智能寻源** - 自动匹配最优供应商和报价
5. **合同管理** - 智能审核和风险识别
6. **成本分析** - 多维度成本分析和优化建议

## 🎨 设计特色

- **靛蓝+琥珀配色** - 体现专业和价值优化
- **供应链可视化** - 直观的网络关系图
- **数据对比** - 清晰的成本和绩效对比
- **Work Sans字体** - 现代专业的企业形象

## 📦 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📈 实际效果

该系统已在某大型制造企业（年采购额50亿+，3000+供应商）全面部署：
- 采购成本: 年度节省9000万+，降本18%
- 采购周期: 从平均45天缩短到27天
- 供应商合格率: 从82%提升至96%
- 采购人员效率: 提升60%，处理订单量翻倍
- 库存周转率: 提升35%，释放资金12亿+

## 🎯 主要特点

- **智能寻源**: AI自动匹配最优供应商组合
- **动态评估**: 供应商实时评分和预警
- **需求预测**: 提前3-6个月精准预测
- **成本优化**: 多维度的采购成本优化方案
