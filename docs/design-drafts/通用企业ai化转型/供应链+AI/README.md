# 链动引擎 ChainPulse - 供应链+AI 智能供应链平台

**客户案例**: 某全球化供应链企业（年营收300亿+）AI供应链管理实践

## 📊 项目概述

链动引擎是一套以智能协同为核心的AI供应链管理平台，通过需求预测、库存优化、物流调度和风险管控，实现供应链的端到端智能化。

### 核心价值

- ✅ **库存成本降低30%** - AI驱动的智能补货和优化
- ✅ **缺货率降低85%** - 精准需求预测和动态调整
- ✅ **物流成本降低22%** - 智能路径规划和运力优化
- ✅ **交付准时率提升至98%** - 全链路可视化和预警

## 🚀 技术栈

- **前端框架**: React 18 + TypeScript + Vite
- **UI组件**: Tailwind CSS + shadcn/ui
- **地图可视化**: Mapbox GL JS + 自定义图层
- **数据可视化**: ECharts + Recharts
- **字体**: Raleway（简洁现代）+ Source Code Pro（数据）

## 💡 核心功能

1. **供应链驾驶舱** - 全链路实时监控和KPI
2. **需求预测** - AI预测市场需求和销售趋势
3. **智能补货** - 自动计算补货点和补货量
4. **库存优化** - 多仓协同和库存平衡
5. **物流调度** - 智能路径规划和运力调配
6. **风险预警** - 供应中断和异常事件预警

## 🎨 设计特色

- **科技蓝+翠绿配色** - 体现智能协同和流动性
- **地图可视化** - 直观的物流网络和仓储分布
- **流程动画** - 动态展示供应链流转
- **Raleway字体** - 现代国际化的视觉风格

## 📦 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📈 实际效果

该系统已在某全球化供应链企业（年营收300亿+，100+仓库，5000+SKU）全面部署：
- 库存成本: 年度节省6亿+，周转率提升40%
- 缺货率: 从15%降至2.5%，客户满意度大幅提升
- 物流成本: 节省超过2亿元，降本22%
- 交付准时率: 从85%提升至98%
- 供应链响应速度: 提升5倍，从72小时到14小时

## 🎯 主要特点

- **全链可视**: 从原料到终端的全程透明化
- **智能预测**: 多维度需求预测和趋势分析
- **动态优化**: 实时调整库存和物流策略
- **风险防控**: 提前识别并应对供应链风险
