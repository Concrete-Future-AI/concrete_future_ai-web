# 财智云 FinanceWise - 财务+AI 智能财务管理平台

**客户案例**: 某大型集团公司（年营收200亿+）AI财务管理实践

## 📊 项目概述

财智云是一套以智能决策为核心的AI财务管理平台，通过自动化记账、智能报表、风险预警和财务预测，实现财务管理的数字化转型。

### 核心价值

- ✅ **财务处理效率提升80%** - AI自动化记账和对账
- ✅ **报表生成时间缩短90%** - 从2天缩短到2小时
- ✅ **风险识别准确率95%** - AI实时监控财务异常
- ✅ **预测准确度92%** - 现金流和收支预测

## 🚀 技术栈

- **前端框架**: React 18 + TypeScript + Vite
- **UI组件**: Tailwind CSS + shadcn/ui
- **数据可视化**: Recharts + ECharts
- **状态管理**: React Hooks
- **字体**: Source Sans Pro（专业稳重）+ Fira Code（财务数据）

## 💡 核心功能

1. **智能看板** - 实时财务状况和关键指标
2. **自动记账** - AI识别发票和自动生成凭证
3. **智能报表** - 一键生成各类财务报表
4. **风险预警** - 实时监控财务风险和异常
5. **现金流预测** - AI预测未来资金状况
6. **税务助手** - 智能税务筹划和申报

## 🎨 设计特色

- **深绿+金色配色** - 体现稳重专业和价值感
- **数据密集型布局** - 清晰的财务信息展示
- **专业图表** - 多维度财务数据可视化
- **Source Sans Pro字体** - 现代专业的视觉体验

## 📦 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
