# 知识宝库 KnowledgeVault - 企业知识库+AI 智能知识管理平台

**客户案例**: 某大型咨询公司（5000+员工）AI知识管理实践

## 📊 项目概述

知识宝库是一套以智能检索为核心的AI知识管理平台，通过文档智能解析、语义搜索、知识图谱和智能问答，让企业知识真正流动起来。

### 核心价值

- ✅ **知识检索效率提升10倍** - 从平均30分钟到3分钟
- ✅ **知识复用率提升200%** - AI推荐相关知识和案例
- ✅ **新员工上手速度提升60%** - 智能培训和知识引导
- ✅ **知识沉淀量提升5倍** - 自动提取和结构化

## 🚀 技术栈

- **前端框架**: React 18 + TypeScript + Vite
- **UI组件**: Tailwind CSS + shadcn/ui
- **图谱可视化**: Cytoscape.js + Force Graph
- **搜索**: Elasticsearch + 向量检索
- **字体**: Nunito Sans（友好易读）+ JetBrains Mono（代码）

## 💡 核心功能

1. **智能搜索** - 语义理解的自然语言搜索
2. **知识图谱** - 可视化的知识关联网络
3. **智能问答** - AI助手秒速回答问题
4. **文档管理** - 自动分类、标签和版本控制
5. **知识推荐** - 基于场景的智能推荐
6. **协作编辑** - 多人实时协作和评论

## 🎨 设计特色

- **紫罗兰+薄荷绿配色** - 体现智慧和清新
- **知识图谱可视化** - 直观的知识网络关系
- **卡片式布局** - 便于浏览和发现
- **Nunito Sans字体** - 友好易读的知识体验

## 📦 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
