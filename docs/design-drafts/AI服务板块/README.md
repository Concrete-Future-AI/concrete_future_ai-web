# AI服务公司官网 - 完整实现文档

## 项目概览

这是一个完整的AI服务公司官网，采用**独特的视觉设计系统**，完全避免了"AI生成"的通病美学。网站采用React + Tailwind CSS技术栈，实现了所有功能和高级交互效果，具有强烈的品牌辨识度。

### 🎨 设计特色
- ✅ **独特字体系统**: Syne + Bitter + Inconsolata（避免Inter/Roboto等常见字体）
- ✅ **琥珀+深蓝灰配色**: 完全避免紫色渐变，建立独特的工业温暖风格
- ✅ **有深度的背景**: 网格纹理、光晕装饰、分层设计
- ✅ **协调的动画系统**: 弹性曲线、交错出场、hover微交互
- ✅ **高转化率文案**: 直接、具体、有承诺、零风险表达

### ⚡ 动画系统详解
**避免生硬的线性动画，使用弹性曲线**

1. **动画曲线**:
   ```css
   /* 弹性曲线 - 自然运动 */
   cubic-bezier(0.34, 1.56, 0.64, 1)
   
   /* 平滑入场 - 优雅流畅 */
   cubic-bezier(0.16, 1, 0.3, 1)
   ```

2. **入场动画**:
   - **fadeIn**: 从60px下方滑入 + 缩放0.95→1 + 模糊10px→0
   - **fadeInUp**: 纯垂直滑入60px
   - **slideInLeft/Right**: 水平滑入80px + 轻微旋转(±5deg)
   - **fadeInScale**: 缩放0.9→1 + 垂直滑入40px

3. **交错时间**:
   - 每个元素延迟增加0.1s（0.15s → 0.95s）
   - 工具卡片按列交错（3列布局）
   - 使用CSS变量 `--item-index` 实现动态延迟

4. **Hover微交互**:
   - 卡片: 上移2px + 边框变琥珀色
   - SVG图标: 旋转6度 + 缩放1.1倍
   - 光晕: 0→20%透明度（blur-3xl）
   - 过渡时间: 300ms
