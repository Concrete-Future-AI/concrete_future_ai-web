# 更新日志

## v2.0 - 独特美学系统 (2025-11-14)

### 🎨 重大设计变更

#### 视觉系统全面升级
- ✅ **字体系统**: 从 Playfair Display + Space Grotesk 更换为 **Syne + Bitter + Inconsolata**
- ✅ **配色方案**: 完全移除紫色渐变，采用 **琥珀橙 + 深蓝灰** 配色
- ✅ **背景设计**: 所有板块添加网格纹理、光晕装饰、分层效果
- ✅ **动画系统**: 从线性过渡改为弹性曲线（cubic-bezier），交错出场

#### 具体变更

**1. 字体替换**
```diff
- Playfair Display (标题)
- Space Grotesk (正文)
- JetBrains Mono (代码)

+ Syne 800 weight (标题) - 几何现代感
+ Bitter 300-700 (正文) - 优雅衬线体  
+ Inconsolata (技术标签) - 等宽字体
```

**2. 配色更新**
```diff
- Indigo + Purple 渐变
- 纯白背景
- Pink/Cyan/Emerald/Orange 引擎色

+ Slate-900 主色（深蓝灰）
+ Amber-500 强调色（琥珀橙）
+ 奶油黄背景 + 网格纹理
+ Amber/Blue/Green/Red 引擎色
```

**3. 背景装饰**
- 添加全局2×2px网格纹理（3%透明度）
- Hero区域：模糊光晕（琥珀+蓝色）+ 40×40px几何网格
- 卡片：顶部渐变线条 + Hover右上角光晕
- 深色板块：琥珀色60×60px网格纹理

**4. 动画升级**
```diff
- 线性过渡: ease-in-out
- 简单滑入: translateY(20px)

+ 弹性曲线: cubic-bezier(0.34, 1.56, 0.64, 1)
+ 复杂入场: translateY(60px) + scale(0.95→1) + blur(10px→0)
+ 交错延迟: 0.15s → 0.95s
+ Hover旋转: rotate(6deg) + scale(1.1)
```

### 🎨 设计原则确立

#### ✅ 应该做的
1. 保持琥珀色系 - 品牌识别核心
2. 使用等宽字体标签 - 技术可信度
3. 添加纹理背景 - 避免平面
4. 使用弹性动画 - 自然运动
5. 文案要具体 - 数字、承诺、时间

#### ❌ 不应该做的
1. 不要用紫色渐变 - AI生成标志
2. 不要纯白背景 - 缺乏深度
3. 不要换回Inter字体 - 失去独特性
4. 不要移除网格纹理 - 变得平庸
5. 不要空洞文案 - "赋能""协同"buzzword
